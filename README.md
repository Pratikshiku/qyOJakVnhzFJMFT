## Law Firm Case Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3af3edb4530b453384fd69f8c83104e7/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users (clients), lawyers, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Lawyer Management
6. Case Source Information Management
7. Case Information Management
8. Case Closure Comment Management
9. Law Firm Opinion Management
10. Lawyer Opinion Management
11. Announcement Information Management

**Lawyer Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Case Source Information Management
6. Case Information Management
7. Case Closure Comment Management
8. Lawyer Opinion Management
9. Announcement Information Management



**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. Lawyer Management
5. Client Management
6. Case Source Information Management
7. Case Information Management
8. Case Closure Comment Management
9. Law Firm Opinion Management
10. Lawyer Opinion Management
11. Basic Data Management (case source type management, announcement type management)
12. Announcement Information Management
